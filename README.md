# jimix

[![Build Status](https://travis-ci.org/manuel-woelker/jimix.svg)](https://travis-ci.org/manuel-woelker/jimix)

A modern JMX web console

## Features

 * Zero third-party dependencies
 * Basic Web UI
 * Simple REST interface (CORS enabled)
